hlp.svc-clust.html HIVE-Octagon Sequence hierarchical clustering HIVE-Octagon allows the user to perform comparative analysis of multiple variant calling (profiling) results. The user has the ability to visualize the hierarchical relationship between SNP frequences in different samples in the form of a downloadable phylogram or NEWICK raw data file.

Parameters may be modified to customize hierarchical clustering of sequences of SNP frequencies for alignment profiling results.


Input Parameters:
    Name

    Specify a filename for the resulting computation.

    Reference genome

    Select the reference genome(s) from which the SNV profiles were generated. More than one reference genome can be selected from the HIVE file explorer by holding Ctrl while selecting individual files. Click Submit to add the selected references to the computation.

    Reference genes

    Select the specific reference gene(s) from your reference genome(s) for which the SNV profiles will be compared. Alignment profiling results for references outside this list will be ignored. If this list is empty, alignment profiling results for all references in the reference genome will be examined.

    HIVE-Heptagon profiling results to compare

    Select the specific profile computations to be included in the comparison. More than one profile can be selected from the HIVE file explorer by holding Ctrl while selecting the individual files.

    Clustering algorithm
    • Neighbor joining — an algorithm by Saitou and Nei that tries to minimize the overall phylogenetic tree length; the algorithm is well-regarded, but slower than single link and complete link.
    • Fast neighbor joining — An algorithm by Elias and Lagergen which is somewhat less accurate than neighbor joining, but much faster on large collections of alignment profiles.
    • Single link — the distance between two clusters of profiles is the minimum of the distances between their included profiles; this algorithm is the fastest for clustering large collections of alignment profiles. Its downside is the "chaining effect", occasionally producing long, unbranched phylogenetic structures.
    • Complete link — the distance between two clusters of profiles is the maximum of the distances between their included profiles; its downside is a high sensitivity to outliers.
