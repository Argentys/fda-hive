hlp.svc-viral-mutation-comp.progress.html Progress Track the status of a job in progress Once submitted, the progress of a given process is reported along with information regarding the submission.

Progress options:
  • Main progress - The default view displays the percent completion and cumulative time elapsed of the total job process as well as the progress for each individual computational node. To view individual results, click on the icon adjacent to the corresponding row. To collapse individual lines, click on the icon.
  • Input objects - This list contains the IDs and names of objects designated as input for this task.
  • Objects using this as input - This list maintains the IDs and names of any objects that may use this computation as an input for further analysis.
  • Info - Any noteworthy information about the completion of the process (for example, errors) can be found here. If the job completes normally, there will be no information displayed.
  • Download - Download the table of computation and wait times for the process.
  • What's next? - This tab lists options for subsequent action using this process.
