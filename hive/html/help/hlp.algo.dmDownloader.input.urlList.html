hlp.algo.dmDownloader.input.urlList url-list input tab</id> <definition>Download data by entering a list of URLs into the text box</definition> <description>The most common way to download the data is to input the URL directly at the <span class="pagetext">url-list</span> tab. The URL may be full path (EX: http://www.ebi.ac.uk/ena/data/view/CAG46501&display=fasta), but it may also be provided without the protocol (EX: www.ebi.ac.uk/ena/data/view/CAG46501&display=fasta). If entered without protocol, the user should specify the data source from which the data will be retrieved using the <span class="pagetext">Datasource</span> select box below. The URL may be the Bioproject ID from NCBI. In this case, the user should specify the data source as <span class="pagetext">NCBI_Bioproject_Nucleotide</span> or <span class="pagetext">NCBI_Bioproject_protein</span> in the <span class="pagetext">Datasource</span> select box and input the ID into the <span class="pagetext">url-list</span> tab. If one or many URLs come from different sources, each must be individually specified (EX: http://, ftp://, etc.. ) or a range must be provided (EX: Bioproject ID range: 59189-59202).</description>
