hlp.svc-recomb.html Reference recombination discovery Identify regions of recombination between closely related references The Reference recombination engine enables concurrent visualization of a single sample aligned to multiple reference segments. By comparing the distribution of sample coverage across different genomes and cross-referencing this coverage to a multiple-alignment of the reference segment, the user can clearly view variable regions of coverage distribution across references, which may imply a recombination event. This tool was specifically designed to facilitate discovery of recombination among closely related viral and bacterial populations.


Selecting Input Data:
    The default input page for the Recombinator is identical to the alignment results output view, with an augmented hit list to enable check box selection of the references to be profiled. References can be selected individually by selecting the checkbox adjacent to the reference of interest or as a complete set by clicking the checkbox in the top left column of the default list alignment results table.

Basic Inputs and Parameters
    Name
    Provide a unique name to the recombination process for easy search and retrieval of results. A default name will be provided if nothing is specified

    Reference Genome Alignment File
    By default, the alignment from which the recombination page was accessed will populate this field. Clicking the dropdown menu will open your user home directory from which you can select alternative alignments as inputs. Alignments can be removed from inputs by selecting the corresponding row and clicking the red icon to the right.
