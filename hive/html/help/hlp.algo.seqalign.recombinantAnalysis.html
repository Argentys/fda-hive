hlp.algo.seqalign.recombinantAnalysis Recombinant analysis Analyze recombination between two reference genes hit in the alignment This section allows the user to perform recombinant analysis between any two of the reference genes listed in pairs tab on the left (corresponding to the genes in the reference genome). The user must select exactly two of these genes by clicking the adjacent checkbox before running the recombinant computation or building a recombinant polyplot (sections to the right).
