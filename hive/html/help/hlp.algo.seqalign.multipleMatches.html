hlp.algo.seqalign.multipleMatches Multiple matches Choose how to view and rank alignment results with multiple matching alignments The alignment results can be filtered by choosing one of three options. The reported matches/alignments can be returned as either the Best match or First match, or as the list of All matches. The First match option can be useful when the user is simply interested in finding the genome to which a query sequence belongs, whereas the /Best match option ranks alignments according to alignment scores.
