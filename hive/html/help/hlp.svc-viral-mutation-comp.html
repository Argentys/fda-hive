hlp.svc-viral-mutation-comp.html Viral Mutation Comparator Compare, view, and filter mutation calls from several sources Mutations reported across several sources can be analyzed, unified and reformatted into a single, easy to read table for further filtration using HIVE's table query capabilities. This allows users to compare previously generated, external mutation calls for viral samples to mutations discovered using HIVE's internal variant calling pipeline. Users may choose to report all mutations across input sources, or choose only to report those which are present in more than one source. A number of follow-up analyses can be performed to mine the newly comprehensive tables for across-sample trends or statistics.
