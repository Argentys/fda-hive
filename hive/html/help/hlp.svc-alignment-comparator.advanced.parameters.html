hlp.svc-alignment-comparator.advanced.parameters.html Alignment Comparator Advanced help for Alignment Comparator Here you can find descriptions of the advanced parameters for the alignment comparator tool.

Advanced Parameters:
    Minimal RPKM
    Specify a minimum number of the reads per kilobase of transcript (reference segment) per million mapped reads required in order to consider a specific reference segment.

    Minimal hits
    Select the minimum number of hits to a reference segment required for inclusion in the computation.

    What to output by
    Specify whether to output the results of comparison with respect to Hits, RPKM, or both.

    Collapse hits by
    Specify to collapse compared alignments to the transcript or gene level by selecting "transcript_id" or "gene_id", respectively.

    Output IDs
    Specify whether the output should be reported as gene or transcript IDs by selecting "gene_name" or "transcript_name", respectively.

    Annotation file for collapsing hits
    Designate an annotation file that provides the gene/transcript names and IDs for your alignments.

    Batch service
    By default, computations are run in single computation mode. Choose "Batch Mode" from the dropdown menu in order to submit alignment comparator processes as batches.
