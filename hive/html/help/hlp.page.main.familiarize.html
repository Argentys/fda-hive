hlp.page.main.familiarize.html Familiarize yourself with HIVE Learn to use and navigate HIVE

Once registered, your browser will automatically route you to your user home page upon login. On this page you may view and manage (add or delete) your available data and processes. All HIVE interfaces employ the same basic organization. Each subsection consists of a viewer and tabular navigation system, with in-page documentation available in adjacent tabs. Two major modes of visualization used by HIVE are the data container and the control list.

The data container model of visualization consists of two main components: tools and content-specific lists. The toolbars for each data container are similar, sharing most utilities with a few specific exceptions. All toolbar functions are listed below.

COMMON TOOLBAR FUNCTIONS:

Refresh the contents of the data container
Cut selected contents from the data container
Paste previously copied contents into selected data container
Hover over or click to view additional editing options
Delete selected objects or folders from the repository
Specify sharing permissions for the selected object or folder
Edit details of the selected object or folder
Undo previous action
Redo previously undone action
In the folder directory or folders tab, create a new folder
View details of the selected object or process (hover over the icon to access detail editing options)
Navigate through paginated tables with entries spanning more than one page
Conduct a text search in the repository
Download the selected object(s) to local user space
Navigate forward (right) to additional tabs in the user home object directory
Navigate backward (left) to additional tabs in the user home object directory

SPECIFIC TOOL SHORTCUTS:

Add a new object using the dmDownloader file loading utility
Jump to the HIVE-Hexagon alignment page
Jump to the page for the HIVESeq sequence manipulation toolkit
Jump to the CensuScope metagenomic analysis tool page

Control lists enable certain data attributes to be user-modified. All control lists contain the following options:

Expand the collapsible heading and add another record
Collapse the associated hierarchy
Expand the associated hierarchy
