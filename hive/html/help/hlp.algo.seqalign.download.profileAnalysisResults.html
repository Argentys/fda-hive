hlp.algo.seqalign.download.profileAnalysisResults Profile Analysis – Download results Download information from hits table as .csv file The alignment results can be downloaded as a .csv file. When the user clicks the download tab, a pop-up will appear prompting the user to open or save the file. After the user makes the appropriate selections the download will proceed. When the file is opened, it will contain all the data from the Table summary with the exception of the corresponding color column and title row.
