hlp.view.summary.snpProfiling SNP Profile Results View statistics of computed SNP profile This is the default view of the SNP profiling results. A table displays quantitative information about the computed SNP profile belonging to three categories. General Information gives the length of the selected reference genome (or consensus) and the number of reference genes involved. Mapped Regions contains information on reads that map to the reference genome. Total Contig Length is the length when all contigs are placed end to end. When there is only one contig (in the case of the reference genome) the contig length will be the length of the entire genome. Mapped Coverage (% Reference) tells the percent of the reference genome that is hit by the selected sequences. Coverage on contigs is the average number of reads per nucleotide in the contigs. Total Number of Contigs is the number of contigs that make up the consensus sequence. Number of Mapped Reads are the number of reads that map to the selected reference or consensus. Similarly, Number of Mapped Positions are the number of individual positions that map to the selected reference or consensus. Nucleotide Matches BP are the number of positions that map to the reference in which the base pair matches the reference. Mismatches (% of Mapped) then are given by the number of mismatches (Number of Mapped Positions - Nucleotide Matches BP) divided by the Number of Mapped Positions. Inserts (% of Mapped) and Deletions (% of Mapped) give the number of inserts and deletions, respectively, among the positions mapped to the reference expressed as a percentage. Unmapped Regions displays information on reads that do not map to the selected reference genome or consensus. Total Length of the Unmapped Regions gives the number of positions in the reference that do not map to the sequence. N Unmapped Regions (% Reference) converts the number of unmapped positions into a percentage of the reference. Coverage on gaps is the average number of reads per position that map to gaps. Total Number of Gaps Found gives the number of positions for which a gap is present in the alignment.
