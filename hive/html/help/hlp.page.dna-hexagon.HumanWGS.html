hlp.dna-hexagon.HumanWGS.html Human Whole Genome Sequencing Preset parameters for HIVE-hexagon alignment of human whole genome sequencing data This page is prepopulated with parameter values appropriate to the scope of human whole genome data. The human genome is 3.3 billion base pairs long, one of the largest genomes. It is structured with DNA base pairs in a diploid organization. The preset inputs on this page are set to optimize the size to accuracy tradeoffs of such large computations.

For basic help using the HIVE-hexagon aligner, see the topic titled HIVE-hexagon. Advanced parameters and help can be found by clicking on the icon found at the top left of the input box.
