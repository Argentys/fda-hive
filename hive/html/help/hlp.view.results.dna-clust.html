hlp.view.results.dna-clust.html Sequence hierarchical clustering results The clustering engine's end output is a tree structure whose leaves are the input data. It is displayed graphically in the form of a phylogram.
Clustering phylogram

This is a graphical view of the hierarchical clustering result. Input items (alignment profiling results) are represented as leaves of the tree, displayed as labeled circles. You can click on the leaf-circles to enable or disable the corresponding SNP frequency graph on the right.

Phylogram branches can be collapsed or expanded by clicking on the inner nodes, represented as squares.

The phylogram can be downloaded in SVG format by clicking on the link below it. Note that some web browsers (e.g. Google Chrome) may refuse to download the file until you click on "keep" or "allow". Others may open the SVG file inside the browser itself, and you will need to use Ctrl-S or File→Save to save it.

Below the phylogram, there are controls to change its shape (circular or rectangular) and the branch scale (choosing "none" makes the phylogram easier to interpret if some branches are much longer than others).

Downloads

Raw output of the hierarchical clustering engine can be downloaded if needed for further analysis.

SNP frequencies

Up to 10 SNP frequency graphs for alignment profiling results (which were selected by clicking leaves in the phylogram or checking items in the Select profilings tab) for the selected reference (chosen in the Select reference tab).

Select profilings

A table of the alignment profiling results which were used as input to the clustering engine. Checking a table item will highlight the corresponding leaf-circle in the phylogram, and will enable the SNP frequency graph in the SNP frequencies tab.

Select references

A table of the references from the reference genome which were used to calculate the clustering. Clicking on a table item will select the corresponding parts of the profiling data for display in the SNP frequencies tab.
