hlp.svc-profiler-heptagon.html HIVE-heptagon profiler Base and variant calling The HIVE-heptagon profiler performs base and SNP-calling for a previously computed alignment and provides quality and noise assessment profiles. Variant-calling is achieved by calculating the frequency of individual bases at specific positions and comparing those values to the base present at the corresponding position in either a reference or consensus sequence. Post-alignment quality control procedures are based on positional base-frequency entropic information to help validate alignment results and distinguish computational artifacts from real biological variability.

Graphical representation of genomic-proteomic mapping is also facilitated through the automatic retrieval of annotations and metadata associated with the input files upon entry into HIVE. Mapping of these annotations to the genomic position and superimposing the results on single graph enables clear visualization of SNP mutations which may affect protein structure.

Basic Inputs and Parameters
The default input page for the SNP profiler displays identical information to the alignment results output summary with the most frequently modified profiling parameters called out below. In this page, however, the list of hits to references is interactive, enabling check box selection of the references to be profiled. Once desired references are selected, you may simply click the "Analyze" button to proceed with default inputs. Additional basic parameters include the following:
  • Name - Provide a unique name to the profiler process for easy search and retrieval of results. A default name will be provided if nothing is specified
  • Profile type - Specify whether profiling is to be computed with respect to the reference sequence provided or the consensus of the reads.
  • Minimal coverage allowed - Specify the minimum coverage of alignment at a position such that any position with coverage less than the designated value threshold will be ignored during the base-calling procedure.
  • Alignments to use - By default, the single alignment from which the profiler page was accessed will populate this field. Clicking the dropdown menu will open your user home directory from which you can select multiple alignments for profiling to the same reference set. Alignments can be removed from inputs by selecting the corresponding row and clicking the red icon to the right.
Note: all parameters have descriptions accessible in the interface by hovering the mouse over a topic.
