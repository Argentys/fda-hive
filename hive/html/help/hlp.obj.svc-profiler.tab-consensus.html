hlp.obj.svc-profiler.tab-consensus.html Consensus View the consensus generated by base-calling of a sample against a reference This tab shows the consensus sequence of reads covering the region of the selected reference segment. The consensus sequence calls the base with highest frequency of occurrence across all reads. The consensus can be generated by treatment of gaps in one of four ways:
  • displayed - Gaps will be maintained and displayed as + at the position.
  • filled - Gap positions will be filled with the reference nucleotide.
  • skipped - Gap positions will be excised and flanking regions will be concatenated.
  • split - Gap positions will be excised and flanking regions will be maintained as separate sequences.
Please note - sequences over a certain length will not appear in their entirety in the window. For these extra long sequences, the user will need to use the horizontal scroll bar to view the whole sequence.
