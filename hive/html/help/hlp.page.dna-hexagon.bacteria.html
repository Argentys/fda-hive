hlp.page.dna-hexagon.bacteria.html Bacterial Genome Sequencing Preset parameters for HIVE-hexagon alignment of bacterial sequences Bacterial genomes are generally smaller than eukaryotic genomes, consisting of a single DNA molecule and thus making bacterial genomes haploid.This page is prepopulated with parameter values optomized for bacterial genome alignment.

In addition to the main HIVE-hexagon parameters (see the topic titled HIVE-hexagon for help with basic HIVE-hexagon functions), the bacterial page has the following specially called out parameter designed specifically to address common properties of bacterial genomes:

Repeat and Transposition discovery - Tell the alignment engine to perform more detailed multi-pass lookups in order to identify repeats and transpositions at the expense of a longer run-time.

Advanced parameters and help can be found by clicking on the icon found at the top left of the input box.
