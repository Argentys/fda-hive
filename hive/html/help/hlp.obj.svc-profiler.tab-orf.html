hlp.obj.svc-proflier.tab-orf.html Open Reading Frame summary View called mutations with respect to annotated open reading frames This tab will display graphical representations of ORFs only if annotations have been supplied. The first graph will show the position of significant SNPs according to the threshold set in the SNP calls tab plotted against their frequencies. The second will show the position of all open reading frames annotated to occur along the length of the specified reference.
