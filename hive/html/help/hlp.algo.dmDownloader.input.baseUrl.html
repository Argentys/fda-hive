hlp.algo.dmDownloader.input.baseUrl Base-URL option Specify base-URL common to list of URLs for data upload If entering a list of several URLs with common path designations, the user can type or copy/paste the base URL into the text box. For example, if the user wants to download the following four files: http://www.pdb.org/pdb/download/downloadFile.do?fileFormat=pdb&compression=NO&structureId=2XXR http://www.pdb.org/pdb/download/downloadFile.do?fileFormat=pdb&compression=NO&structureId=2XXT http://www.pdb.org/pdb/download/downloadFile.do?fileFormat=pdb&compression=NO&structureId=2XXU http://www.pdb.org/pdb/download/downloadFile.do?fileFormat=pdb&compression=NO&structureId=2XXV the user can input “http://www.pdb.org/pdb/download/downloadFile.do?fileFormat=pdb&compression=NO&structureId=” in the Base URL text box and input “2XXR,2XXT,2XXU,2XXV” using the url-list tab above.
