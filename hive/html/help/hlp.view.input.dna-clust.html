hlp.view.input.dna-clust.html Sequence hierarchical clustering algorithm input data
Alignment profiling results to compare

A list of alignment profiling results. These must all be generated from alignments against the same reference genome (but possibly against different references within that reference genome). The sequences of SNP frequencies from the selected alignment results serve as the main input to the hierarchical clustering algorithm.

Reference genome

A genomic sequence; all of the chosen alignment profiling results must be generated from alignments against this sequence.
