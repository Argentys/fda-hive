hlp.view.histogram.qualityHisto Query Sequence quality-histogram View a histogram showing the average quality score as a function of sequence position This tab is only present when an individual sequence (with green check mark icon) is selected from the Query Sequences (reads) menu. Clicking this tab displays a histogram with sequence position on the X axis and the quality of the corresponding position on the Y axis. All quality values are reported as Phred scores. When the user hovers over each position, a small pop-up window displays the exact quality of the selected position.
