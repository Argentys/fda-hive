hlp.svc-alignment-comparator.html Alignment Comparator General help for Alignment Comparator The Alignment Comparator is a tool that enables comparison of many individual reference-guided alignments with respect to certain features.

General Parameters:
    Name
    Here you can designate a name for this specific process for simple search and retrieval from your user home directory.

    Alignments to use
    Select the alignments of interest from the dropdown menu. Ctrl+click to select multiple alignments at once. When finished selecting, please click "Submit".

    Classify partial subalignments
    Partial subalignments can be ignored as indicated by the default "Do not classify", or "Split into subalignment categories" which will create new categories for each subalignment.
