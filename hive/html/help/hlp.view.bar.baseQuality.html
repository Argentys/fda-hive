hlp.view.bar.baseQuality Query Sequence base-quality bar chart View the average quality for each base in the query sequence as a bar chart This tab is only present when an individual sequence (with green check mark icon) is selected from the Query Sequences (reads) menu. Clicking on this tab displays a bar graph with a bar for each nucleotide on the X axis and the average quality score of the corresponding nucleotide on the Y axis. All quality values are reported as Phred scores. When the user hovers over a bar, a small pop-up window shows the exact average quality score of the selected nucleotide.
