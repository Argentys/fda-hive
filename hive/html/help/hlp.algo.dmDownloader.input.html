hlp.algo.dmDownloader.input Input box Enter location from which to retrieve your data and specify parameters for your download This box contains several options allowing the user flexibility when uploading data. Four tabs can be accessed to upload the file in the form of a url-list, local-file, url-file or url from a remote server. Additional options allow the user to specify a base URL, the datasource from which the files will be retrieved and the level of concurrent downloads.
