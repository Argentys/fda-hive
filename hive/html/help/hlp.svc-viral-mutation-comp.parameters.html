hlp.svc-viral-mutation-comp.parameters.html Input parameters Basic input required Here you find the most frequently modified parameters and the minimal inputs required to run the viral mutation comparator tool. Additional optional algorithm and system parameters can be found by opening the corresponding tabs. Short descriptions for parameters can be found by hovering the mouse over the input field. Additional information is included in the text and links below.

Basic inputs and parameters:
  • Name - Choose a name for this specific comparison process for simple search and retrieval from your user home directory.

  • Mutation calls from alternative pipelines - Supply mutation data from results of an external analysis, for example, data reported in support of a submission. Externally generated mutation data must be very precisely organized in order to be properly parsed and used by HIVE's table query engine. The supplied table file should be in a recognizable table format (.csv, .xls) and should include the following columns, with headings exactly as they appear below, and in the same order as they appear below (from left to right):

    USUBJID Subject ID, for example, a clinical trial patient identifier
    NGSPL Technological platform used to generate NGS sample data
    VISIT Reason for sample collection, for example, baseline, or follow-up
    AAPOS Amino acid position of the called mutation as defined by position on reference genome
    TCOV Total experimental coverage of sequencing data at the position of interest
    AAREF Amino acid called at the position of interest in the reference genome, designated by its single letter amino acid abbreviation
    AASUB Substituted amino acid called at the position in the experimentally identified mutant, designated by its single letter amino acid abbreviation
    VCOV Coverage of mutant calls at the position of interest
    AAFREQ Frequency of occurrence of the mutant, equal to VCOV/TCOV

    For your convenience, we have supplied a .csv template that can be downloaded by clicking the following link.

    Viral Mutation Comparator Template Table

  • HIVE heptagon variants - Supply mutation data directly from a HIVE-heptagon variant call file by choosing the appropriate HIVE object from your user home directory. Profiles must be run in a specific way in order to be compatible with the Viral Mutation Comparator tool. In theadvanced parameters section of theHIVE-heptagon profilerthe option for "Compute AA profile" must be enabled by selecting "yes" from the dropdown menu. This parameter setting tells the algorithm to compute not only the nucleotide profile, but to convert bases by codon into their corresponding amino acids. By default, this parameter is set to "no". Once submitted, a profiler computation object will appear in the computations tab of your user home directory and will be available for selection as an input for the Viral Mutation Comparator tool.
