hlp.view.dataContainer.html Data container visualization Definition of data container visualization The data container model of visualization consists of two main components: the tools and content-specific lists. The toolbars for each data container are similar, sharing most utilities except for a few specific exceptions. All toolbar functions are listed below:



COMMON TOOLBAR FUNCTIONS:

Refresh the contents of the data container

Cut a folder to access the copy option hover on the cut option
Copy a folder to access this option hover on the cut option
Paste an object that has been cut or copied to a different location
Download the contents of the selected element to the local computer

Delete selected objects from the repository

Edit content of selected object

Browse informartion about selected objects \
Specify sharing permissions for the selected object

Navigate through paginated tables with entries spanning more than one page

Conduct a text search in the repository



SPECIFIC TOOLBAR FUNCTIONS:

Jump to HIVE-Hexagon alignment utility

Jump to Hiveseq utility
