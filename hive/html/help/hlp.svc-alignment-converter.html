hlp.svc-alignment-converter.html DNA Alignment Converter General help for DNA Alignment Converter DNA Alignment Converter is a tool that converts SAM files into a HIVE alignment computation for ease of use in downstream processing using HIVE tools. DNA Alignment Converter uses the SAM file and reference genome(s) used to generate the SAM file as inputs.

Basic inputs and parameters:
  • Name - In this field, you can designate a name for the generated HIVE alignment computation for simple search and retrieval from your user home directory.
  • Source SAM file - Select the SAM file you are interested in convertnig into a HIVE alignment computation format.
  • Reference Genome - Select the genome(s) that were used in the alignment to produce the SAM file you are converting. Multiple reference genomes can be selected in the file explorer by holding Ctrl and clicking each reference.
